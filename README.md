Данный проект состоит из пяти классов: Main, DatabaseManager, MyGraphics, Laval_Nozzle, oxygenTueyre. 
Описание классов:
- oxygenTueyre - класс для расчета кислородной фурмы: количесвта сопел и угла наклона оси сопла к вертикали;
- Laval_Nozzle - класс для расчета параметров сопла Лаваля;
- MyGraphics - класс для генерации примитивного чертежа по рассчитанным параметрам сопла Лаваля;
- DatabaseManager - класс для создания и использования базы данных (база данных создана посредством sqlite);
- Main - класс для графического интерфейса, основной класс программы.
